The Art of
Rotoscoping

Rotoscoping is the process of tracing over live-action footage, frame by frame, to produce realistic animation. I sourced a YouTube video and methodically traced each frame — layering hand-drawn lines over real movement to create a fluid, stylized animation. The result bridges realism and artistry in a way that purely digital animation cannot replicate.

This project reflects my passion for animation and my growing skills in multimedia production. Through this process, I explored how technology and creativity work together to produce engaging visual content.

Solar System
in Blender

To support and enhance the rotoscoping project, I built a fully animated solar system in Blender using a custom Python script. Each planet orbits the sun with accurate relative speeds, textures, and lighting — rendered with Cycles for photorealistic output. This component demonstrates 3D animation fundamentals while tying directly into the cosmic visual theme.

Python scripting in Blender allowed me to automate planet placement, orbital paths, and keyframe generation — drastically reducing manual work and allowing rapid iteration on the animation timeline.

🌐
Scene Setup
Created sun, planet meshes, and HDRI space environment inside Blender's 3D viewport.
🐍
Python Scripting
Automated orbital keyframes, planet speeds, and lighting setup with a custom bpy script.
💡
Lighting & Materials
Applied emission shaders to the sun, PBR planet textures, and atmospheric rim lighting.
🎬
Render & Composite
Rendered with Cycles engine, composited glow effects, and exported for final integration.
solar_system.py
# Solar System — Blender Python Script import bpy, math planets = [ ("Mercury", 0.4, 2.4, 88), ("Venus", 0.9, 6.0, 225), ("Earth", 1.0, 8.0, 365), ("Mars", 0.5, 10.5, 687), ] for name, size, dist, period in planets: bpy.ops.mesh.primitive_uv_sphere_add(radius=size) planet = bpy.context.object planet.name = name # Animate orbit over 250 frames for frame in range(0, 251): angle = (2 * math.pi * frame) / period planet.location.x = dist * math.cos(angle) planet.location.y = dist * math.sin(angle) planet.keyframe_insert("location", frame=frame)
